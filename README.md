SmartLibraryPlus, **Hibernate ORM** ve **SQLite** kullanan, konsol tabanlı bir **kütüphane yönetim sistemi** uygulamasıdır. Öğrenciler, kitaplar ve ödünç alma işlemlerini yönetmek için geliştirilmiştir.

  Özellikler:
*  Kitap ekleme, listeleme, güncelleme ve silme
*  Öğrenci ekleme ve yönetimi
*  Kitap ödünç alma ve iade işlemleri
*  Hibernate ORM ile veritabanı yönetimi
*  SQLite veritabanı desteği

  Kullanılan Teknolojiler:
* Java
* Hibernate ORM
* SQLite
* Maven

  Kurulum ve Çalıştırma:
1. Bu projeyi klonlayın veya indirin.
2. Projeyi bir IDE'de açın (IntelliJ IDEA, Eclipse, VS Code vb.).
3. Maven bağımlılıklarının otomatik olarak indirilmesini bekleyin.
4. `Main.java` dosyasını çalıştırın.

 Kullanım:
Uygulama çalıştırıldığında konsol üzerinden bir menü sunar. Menü yardımıyla:
* Kitap işlemleri
* Öğrenci işlemleri
* Ödünç alma / iade işlemleri
  kolayca yapılabilmektedir.
